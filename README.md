# Research

A place for some experiments and mini research projects

## Projects

- `cf-preview-subfolder`: A small Cloudflare Pages preview experiment focused on serving and validating preview HTML content in a nested project structure. See the project README: [`cf-preview-html/README.md`](cf-preview-html/README.md).
- `dm-video-player`: A Web Component wrapper around Adobe Dynamic Media's video player, with a GitHub Pages subpage published from `docs/dm-video-player/`.
- `shoppableimage`: A planning note for a shoppable image built as an EDS block enhanced by a web component, with metadata from AEM Content Fragments and image delivery through AEM and Dynamic Media. See the project README: [`shoppableimage/README.md`](shoppableimage/README.md).
