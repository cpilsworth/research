# oidc-edge-gate

> **This project has moved to its own repository:**
> **https://github.com/cpilsworth/oidc-edge-gate**

An [AEM Edge Function](https://experienceleague.adobe.com/en/docs/experience-manager-cloud-service/content/implementing/developing/edge-functions) (Fastly Compute) that fronts an [Edge Delivery Services](https://www.aem.live/) site as an OpenID Connect relying party, enforcing access via a three-tier path policy (`public` / `protected` / `secured`).

The full source, tests, and design docs were extracted from this monorepo into the standalone repo above (history preserved) on 2026-06-11. Please open issues and pull requests there.
