Web Component

Adobe Dynamic Media
Video Player

A lightweight custom element that embeds the Adobe Dynamic Media HTML5 player runtime directly—no iframe required.

Live Demo

Usage

<script type="module" src="/path/to/adobe-dm-video-player.js"></script> <adobe-dm-video-player src="https://delivery-….adobeaemcloud.com/adobe/assets/urn:…/play" controls ></adobe-dm-video-player>

Attributes

src Required. The Dynamic Media /play URL for the asset.
controls Show player controls. Defaults to true.
autoplay Start playback automatically on load. Boolean attribute.
token Secure-delivery token. Falls back to any token query param already present in src.
letterboxed Preserve the video's native aspect ratio with letterboxing. Defaults to false.
proxy-base-path Same-origin proxy base path to route manifests and media segments through your server.
