Web Component

Adobe Dynamic Media
Video Player

A lightweight custom element that embeds the Adobe Dynamic Media HTML5 player runtime directly—no iframe required.

Live Demo

Usage

<script type="module" src="/path/to/adobe-dm-video-player.js"></script> <adobe-dm-video-player src="https://delivery-….adobeaemcloud.com/adobe/assets/urn:…/play" controls ></adobe-dm-video-player>

Attributes

src Required. The Dynamic Media /play URL for the asset.
controls Show player controls. Defaults to true.
autoplay Start playback automatically on load. Boolean attribute.
token Secure-delivery token. Falls back to any token query param already present in src.
letterboxed Preserve the video's native aspect ratio with letterboxing. Defaults to false.
proxy-base-path Same-origin proxy base path to route manifests and media segments through your server.
styles Raw CSS string automatically scoped to this player instance. Rules are prefixed with the container ID so they never affect other players. Also settable via the customStyles JS property.
stylesheet URL of an external CSS file to load. The stylesheet is global (affects all instances) but is reference-counted and removed from the document when the last instance that requested it disconnects.

Styling

--vjs-control-bar-bg Control bar background color.
--vjs-progress-color Played-progress fill color.
--vjs-load-progress-color Buffered-progress fill color.
--vjs-big-play-bg Big play button background color.
--vjs-big-play-border Big play button border color.
--vjs-big-play-color Big play button icon color.
--vjs-volume-bar-color Volume level fill color.
--vjs-control-color Control bar button icon color.
<!-- CSS custom properties (set inline or via external CSS) --> <adobe-dm-video-player src="…" style="--vjs-progress-color: #f4562d; --vjs-big-play-bg: #f4562d;" ></adobe-dm-video-player> <!-- Inline scoped CSS (auto-prefixed to this instance's container) --> <adobe-dm-video-player src="…" styles=".vjs-big-play-button { border-radius: 50%; }" ></adobe-dm-video-player> <!-- External stylesheet (loaded once, ref-counted across instances) --> <adobe-dm-video-player src="…" stylesheet="/styles/player-theme.css" ></adobe-dm-video-player>

Styled Examples

Theme the player with CSS custom properties on the style attribute, or use the styles attribute for scoped CSS rules that are automatically prefixed to the instance.

Branded — orange controls

<adobe-dm-video-player src="…/play" style=" --vjs-control-bar-bg: #f4562d; --vjs-progress-color: #fff; --vjs-load-progress-color: rgba(255,255,255,0.3); --vjs-big-play-bg: #f4562d; --vjs-big-play-border: transparent; --vjs-big-play-color: #fff; --vjs-control-color: #fff; --vjs-volume-bar-color: rgba(255,255,255,0.7); " ></adobe-dm-video-player>
