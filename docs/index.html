Research Project Index

Mini research projects and experiments

This site collects small experiments from the repository and gives each one its own published page when it makes sense to explore interactively.

Dynamic Media Video Player

A Web Component wrapper around Adobe Dynamic Media's HTML5 video player so it can be embedded directly into a host page without an iframe.

Includes a live demo, usage notes, supported attributes, implementation details, and a summary of the main tradeoffs compared with the standard iframe approach.

Content Fragment HTML Preview

A small Cloudflare Pages-oriented preview experiment for generating Handlebars-based HTML previews for AEM Content Fragments.

Repository location: cf-preview-html/. This project currently documents its behavior in the repo rather than through a dedicated published subpage.
